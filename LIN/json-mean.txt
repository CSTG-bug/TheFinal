| 字段名          | 含义            | 说明                  |
| ------------ | ------------- | ------------------- |
| `model`      | 模型名称          | LIN / Ridge / Lasso |
| `cv_r2_mean` | 五折交叉验证 R² 平均值 | 反映训练稳定性与泛化能力        |
| `cv_r2_std`  | 五折 R² 标准差     | 越小表示模型稳定性越高         |
| `train_r2`   | 训练集 R²        | 拟合程度（越接近 1 越好）      |
| `train_mape` | 训练集 MAPE (%)  | 平均百分比误差             |
| `test_r2`    | 测试集 R²        | 模型在未见数据上的表现         |
| `test_mape`  | 测试集 MAPE (%)  | 越小越好                |
| `test_mae`   | 测试集 MAE       | 平均绝对误差（单位同目标量）      |
| `test_rmse`  | 测试集 RMSE      | 均方根误差（单位同目标量）       |
